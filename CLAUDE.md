# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

NLH（No-Limit Hold'em）ポーカー戦術ガイドの単一Markdownドキュメント（`nlh_poker_tactics.md`）を管理するリポジトリ。コードは存在しない。

## ドキュメント構成

`nlh_poker_tactics.md` は以下の8章構成：

| 章 | 内容 |
|---|---|
| 第1章 | プリフロップ戦略（ポジション・3bet・スクイーズ等） |
| 第2章 | フロップ戦略（テクスチャ別CB判断） |
| 第3章 | ターン・リバー戦略（ディレイCB・プローブ・チェックレイズ） |
| 第4章 | オッズ・エクイティ（ポットオッズ・インプライドオッズ・EQ実現性） |
| 第5章 | GTOの基本理念（MDF・混合戦略） |
| 第6章 | レンジ思考とベットサイジング（ポラライズ・ジオメトリック） |
| 第7章 | エクスプロイト戦略（リーク対応・スタット読み・マルチウェイ） |
| 第8章 | トーナメント固有の調整（ICM・バブル・FT戦略） |

## 執筆スタイルの原則

- 各節の構成：**理論的根拠 → 代表例 → 変数と傾向変化**
- ベースライン：キャッシュゲーム 100BB / 6max
- トーナメント固有の差異は第8章にのみ記載
- 主な参照元：Amu 氏（Note記事）、前元氏、松尾氏
- 説明は日本語、技術用語は英語併記（例：「ポラライゼーション（Polarization）」）
