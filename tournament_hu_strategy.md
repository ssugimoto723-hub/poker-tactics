# トーナメント ヘッズアップ戦略

> **ベースライン**: MTT / SNG のヘッズアップ（残り2人）。スタック表記は BB 数。  
> **凡例（オープン）**: `■` = レイズオープン　`△` = 混合（レイズまたはリンプ）　空白 = フォールド  
> **凡例（BBディフェンス）**: `○` = コール　`◆` = 3betブラフ　`○◆` = 3betバリュー　`△` = 混合　空白 = フォールド

---

## 1. HUの基本原則

### ハンド評価の変化

6max との最大の違いは**相手が1人しかいない**こと。ハンドの絶対的な強さより**相対的な強さ**が支配的になる。

| 6max での評価 | HU での評価 |
|-------------|-----------|
| 中程度（KTo、J8s など）| 強ハンド扱い |
| 弱い（K2o、Q6o など）| ミディアム扱い |
| 非常に弱い（72o、83o）| それでも争う価値あり |

相手のレンジが幅広くなるため、**トップペアより弱いハンドでもバリューを取れる**場面が多くなる。

### ポジションとアグレッションの重要性

HU ではポジションが固定される。

- **SB（スモールブラインド）**: プリフロップは後攻、ポストフロップは先攻（OOP）
- **BB（ビッグブラインド）**: プリフロップは先攻、ポストフロップは後攻（IP）

OOP のSBが不利を補うために必要なのが**アグレッション**。パッシブな SB プレイヤーは長期的に必ず負ける。ベット・レイズで情報を取りに行く姿勢が不可欠。

## 2. プリフロップ戦略

### SBのアクション選択

SB は主に3つの選択肢を持つ。

| アクション | 概要 | 使い分け |
|-----------|------|---------|
| **ミニレイズ（2bb）** | 最も一般的。リスクを抑えてポットを作れる | メインの戦略 |
| **リンプ（1bb）** | 弱いハンドでポストフロップを安く確認する | 深いスタック（30BB+）で一部使用 |
| **フォールド** | 期待値がマイナスの弱いハンドのみ | 全ハンドの約25〜35% |

**なぜリンプを使うか（深いスタック時）**  
非常に弱いハンドをレイズすると BB の 3bet に対して降りざるを得ず損失が大きくなる。リンプにより BB のチェック/ベット反応を見てからフロップで判断できる。ただし弱すぎるリンプは相手に安易にベットされ続けるため注意。

### BBのアクション選択

SBのレイズに対して BB は3つの選択肢を持つ。

| アクション | 使うタイミング |
|-----------|-------------|
| **3bet バリュー** | AA〜QQ、AKs/o など超強ハンド |
| **3bet ブラフ** | A2s〜A5s など（ブロッカー価値） |
| **コール** | バリューとブラフ以外の広い範囲 |
| **フォールド** | 最弱のオフスート（72o 等）のみ |

## 5. フロップ〜リバーの考え方

### CB頻度はHUで高くなる

相手1人だけのレンジに対して CB を打てる条件が揃いやすい。6max ではフロップ CB 頻度が50〜60%程度だが、HU では70〜80%に上がることが多い。

| 状況 | HU での推奨 |
|------|-----------|
| 相手がチェックしてきた | 高頻度でベット（相手のチェックレンジは弱い） |
| 相手がベットしてきた | コールレンジを広く取る（ブラフキャッチ重視） |
| ナッツ系ハンド | スロープレイより積極的なバリューベット |

### ブラフの頻度と選定

HU のブラフは 6max より**直接的**に選べる。相手の1つのレンジに対してフォールドを取れるかを考えればよい。

- ターン・リバーのブラフ：相手がショーダウンバリューのあるハンドを切れないようなボードで使う
- セミブラフ（ドロー系）：相手の弱いペアに対してフォールドエクイティ＋アウツを持てる状況

## 6. スタック深度別の戦略変化

| 残スタック | 戦略の軸 | ポイント |
|----------|---------|---------|
| 30BB以上 | 通常のHU戦略 | リンプ/ミニレイズのミックス、ポストフロップ重視 |
| 15〜30BB | アグレッシブなプリフロップ | 3betがオールイン圏に入り始める。ポジション不利な SB のリンプが減る |
| 10〜15BB | プッシュ/3betオールイン移行 | ミニレイズからの3betはほぼオールイン。コール/フォールドが主な判断 |
| 10BB以下 | 純粋プッシュ/フォールド | Nash 均衡 HU チャートを参照。SB はほぼ全ハンドプッシュ |

### 10BB以下のHU Nash均衡（目安）

SBは**ほぼ全ハンドをプッシュ**（フォールドは非常に限られる）。BBのコールレンジは残スタックに応じて変動。

| BB のスタック | コール目安 |
|------------|----------|
| 10BB | ~45%（KTo+、A7o+、22+ 相当） |
| 8BB | ~50%（K8o+、A5o+、22+ 相当） |
| 6BB | ~60%（Q8o+、A3o+、22+ 相当） |
| 4BB | ~75%（ほとんどのブロードウェイ、ペア） |

## 7. 3人→2人への移行期

残り3人（3-handed）から HU に移行する局面は ICM 的に非常に重要。

### 移行前（3人残り）の注意

- 相手2人のスタック差に注意する。ショートスタックが先にバストアウトするほど自分の $EV が上がる
- 自分が脱落するリスクを避けつつ、ショートを攻め立てる
- ミドルスタックはリスクを取りすぎない（バブルに近い感覚）

### 移行後（HU突入直後）の注意

- HU では ICM のプレッシャーが**大幅に減少**する（1位か2位しかないため）
- 賞金差が大きい場合は保守的になる傾向があるが、それが必ずしも正しいわけではない
- スタック差が2対1以上ある場合はビッグスタック側が有利。押し込み戦略が有効

### スタック差別の方針

| スタック状況 | 方針 |
|-----------|------|
| 2:1以上の差（自分がビッグ） | 積極的にプレッシャーをかけ続ける。相手がフォールドを強いられる |
| ほぼ同等スタック | 標準 HU 戦略。スキルと判断力の勝負 |
| 2:1以上の差（自分がショート） | プッシュ/フォールドに近い戦略。有利な状況で積極的にダブルアップを狙う |
